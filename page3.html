FRÅGOR C: CSS

1. Vad är en regel i CSS och vad består den av?

En regel i CSS är en stilinstruktion som beskriver hur vissa element ska se ut. I en regel finns en del som identifierar elementet (selector) och sedan en del som förklarar hur det ska se ut (declaration).

2. Vad menas med kaskad i CSS?

Ordet kaskad refererar till hur stilinformation i CSS kan komma från flera olika källor som bygger på varandra där den med mest prioritet eller specificering i CSS vinner. J Robbins (2018) liknar det vid en kaskad, ett vattenfall i flera steg, där en vattendroppe fortsätter nedåt och nedåt fram tills annat vatten går före.

Källa: Jennifer Niederst Robbins (2018) "Learning Web Design - A beginners guide to HTML, CSS, Javascript, and web graphics", Femte upplagan, O'Reilly
